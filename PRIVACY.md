# Nuancera Privacy Policy

Last updated: June 25, 2026

Nuancera is developed by Duo CAI. This policy explains what information Nuancera handles and how it is used.

## Summary

Nuancera is a local-first desktop application. It does not require an account, does not upload user files, and does not include analytics, advertising, or telemetry.

## Information Nuancera Handles

Nuancera may handle the following information on your device:

- Palettes, color selections, project settings, language preference, and update reminder preferences saved by the app.
- PDF files, images, or other local files that you choose to open in the app.
- Exported files that you choose to create and save.

This information is processed locally on your device and is not sent to Duo CAI.

## Update Checks

Nuancera may check GitHub Releases to see whether a newer version is available. This request does not include your documents, palettes, or local files. GitHub may process standard network information such as IP address and user agent according to GitHub's own privacy policy.

If you choose to download an update, Nuancera may open the release page in your browser or direct you to the update file.

## Microsoft Store

If you install Nuancera from Microsoft Store, Microsoft may process store, purchase, installation, crash, or device information according to Microsoft's own privacy policies and Store terms. Duo CAI does not control Microsoft Store account or platform data.

## Local Storage

Nuancera stores app data locally in the operating system's application data folder. Removing the app may not automatically delete all local user data created by the app or exported by you.

## Third-Party Services

Nuancera does not use third-party analytics, advertising networks, or cloud storage. The only network service used by the app itself is GitHub Releases for optional update checks.

## Children's Privacy

Nuancera is not directed to children and does not knowingly collect personal information from children.

## Changes

This privacy policy may be updated when Nuancera's features or data handling practices change. The updated policy will be published in this repository.

## Contact

For privacy questions, contact the developer through the Nuancera GitHub repository:

https://github.com/Chase-DC/nuancera/issues

# Nuancera 隐私政策

最后更新：2026 年 6 月 25 日

Nuancera 由 Duo CAI 开发。本政策说明 Nuancera 会处理哪些信息，以及这些信息如何被使用。

## 摘要

Nuancera 是一个本地优先的桌面应用。它不要求创建账户，不上传用户文件，也不包含分析、广告或遥测功能。

## Nuancera 处理的信息

Nuancera 可能在你的设备上处理以下信息：

- 应用保存的调色板、颜色选择、项目设置、语言偏好和更新提醒偏好。
- 你主动在应用中打开的 PDF、图片或其他本地文件。
- 你主动创建并保存的导出文件。

这些信息都在你的设备本地处理，不会发送给 Duo CAI。

## 更新检查

Nuancera 可能会读取 GitHub Releases，以检查是否存在新版本。这个请求不包含你的文档、调色板或本地文件。GitHub 可能会根据其隐私政策处理常规网络信息，例如 IP 地址和 user agent。

如果你选择下载更新，Nuancera 可能会在浏览器中打开 release 页面，或引导你访问更新文件。

## Microsoft Store

如果你通过 Microsoft Store 安装 Nuancera，Microsoft 可能会根据其隐私政策和商店条款处理商店、购买、安装、崩溃或设备相关信息。Duo CAI 不控制 Microsoft Store 账户或平台数据。

## 本地存储

Nuancera 会在操作系统的应用数据目录中保存应用数据。卸载应用不一定会自动删除所有由应用创建或由你导出的本地用户数据。

## 第三方服务

Nuancera 不使用第三方分析、广告网络或云存储。应用自身使用的唯一网络服务是用于可选更新检查的 GitHub Releases。

## 儿童隐私

Nuancera 并非面向儿童的应用，也不会有意收集儿童的个人信息。

## 政策变更

当 Nuancera 的功能或数据处理方式发生变化时，本隐私政策可能会更新。更新后的政策会发布在本仓库中。

## 联系方式

如有隐私相关问题，请通过 Nuancera GitHub 仓库联系开发者：

https://github.com/Chase-DC/nuancera/issues
